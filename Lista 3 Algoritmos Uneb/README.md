Este repositório contém diversas soluções possíveis para os problemas apresentados na Lista 3 de algoritmos da Universidade Estadual Da Bahia (UNEB). Os algoritmos foram programados em linguagem C. A estrutura lógica dos códigos se resume em: If Else chain, while / do while / for, além de switch case e vetores.

Para utilzar, deve-se compilar os códigos antes. O compilador utilizado para a programação dos algoritmos foi o GCC. E a IDE foi o Dev-C++ e o Visual Studio Code.
